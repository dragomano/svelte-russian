Мы рекомендуем использовать [SvelteKit](/kit/getting-started/introduction), который позволяет [создавать практически что угодно](/kit/getting-started/project-types). Это официальный фреймворк для разработки приложений от команды Svelte, работающий на базе [Vite](https://vite-docs.ru/). Создайте новый проект с помощью команды:

import { Tabs, TabItem } from '@astrojs/starlight/components';

<Tabs syncKey="starlight-package-managers-pkg">
  <TabItem icon="seti:npm" label="npm">
    ```sh
    npx sv create myapp

    npm install
    npm run dev
    ```

  </TabItem>
  <TabItem icon="pnpm" label="pnpm">
    ```sh
    pnpx sv create myapp
    cd myapp
    pnpm install
    pnpm run dev
    ```
  </TabItem>
  <TabItem icon="seti:yarn" label="yarn">
    ```sh
    yarn dlx sv create myapp
    cd myapp
    yarn install
    yarn dev
    ```
  </TabItem>
</Tabs>

Не беспокойтесь, если вы ещё не знакомы со Svelte! Вы можете игнорировать все замечательные функции, которые SvelteKit предлагает на данный момент, и вернуться к ним позже.

## Альтернативы SvelteKit

Вы также можете использовать Svelte напрямую с Vite через [vite-plugin-svelte](https://github.com/sveltejs/vite-plugin-svelte): запустите `npm create vite@latest` и выберите вариант `svelte` (или добавьте плагин в `vite.config.js`, если проект уже существует). При сборке (`npm run build`) в папке `dist` появятся готовые HTML, JS и CSS файлы. В большинстве случаев вам также понадобится [выбрать библиотеку маршрутизации](https://svelte.dev/packages#routing).

:::note
Vite часто используется в автономном режиме для сборки [одностраничных приложений (SPA)](https://svelte.dev/docs/kit/glossary#SPA), которые также можно [создавать с помощью SvelteKit](/kit/build-and-deploy/single-page-apps).
:::

Существуют также плагины для [Rollup](https://github.com/sveltejs/rollup-plugin-svelte), [Webpack](https://github.com/sveltejs/svelte-loader) [и некоторых других сборщиков](https://sveltesociety.dev/packages?category=build-plugins), но мы рекомендуем Vite.

## Инструменты для редакторов

Команда Svelte поддерживает [расширение для VS Code](https://marketplace.visualstudio.com/items?itemName=svelte.svelte-vscode), а также существуют интеграции с различными другими [редакторами](https://sveltesociety.dev/collection/editor-support-c85c080efc292a34) и инструментами.

Вы можете проверять свой код из командной строки, используя команду [`npx sv check`](/cli/commands/sv-check/).
