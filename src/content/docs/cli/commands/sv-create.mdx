`sv create` создаёт новый проект SvelteKit с возможностью [настройки дополнительных функций](/cli/commands/sv-add#официальные-дополнения).

## Параметры

### `--template <название>`

Выбрать шаблон проекта:

- `minimal` — минимальная заготовка для нового приложения
- `demo` — демо-приложение с игрой в угадывание слов, работающей без JavaScript
- `library` — шаблон для библиотеки Svelte с настройкой `svelte-package`

### `--types <вариант>`

Настроить проверку типов в проекте:

- `ts` — по умолчанию для файлов `.ts` и компонентов `.svelte` с использованием `lang="ts"`
- `jsdoc` — применяет [синтаксис JSDoc](https://www.typescriptlang.org/docs/handbook/jsdoc-supported-types.html) для типов

### `--no-types`

Отключить проверку типов. Не рекомендуется!

### `--no-add-ons`

Выполнить команду без интерактивного запроса дополнений

### `--no-install`

Пропустить установку зависимостей
