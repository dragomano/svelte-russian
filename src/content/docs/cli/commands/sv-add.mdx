`sv create` добавляет новую функциональность в существующий проект.

## Использование

import { Tabs, TabItem } from '@astrojs/starlight/components';

    ```sh

    ```
  </TabItem>
  <TabItem icon="pnpm" label="pnpm">

## Параметры

### `-C`, `--cwd`

Путь к корневой папке проекта Svelte(Kit)

### `--no-preconditions`

Пропустить проверку предварительных условий

### `--no-install`

Пропустить установку зависимостей
