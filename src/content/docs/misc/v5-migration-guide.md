–í–µ—Ä—Å–∏—è 5 –≤–∫–ª—é—á–∞–µ—Ç –≤ —Å–µ–±—è –æ–±–Ω–æ–≤–ª—ë–Ω–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –∏ —Å–∏—Å—Ç–µ–º—É —Ä–µ–∞–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏. –•–æ—Ç—è –Ω–∞ –ø–µ—Ä–≤—ã–π –≤–∑–≥–ª—è–¥ —ç—Ç–æ –º–æ–∂–µ—Ç –ø–æ–∫–∞–∑–∞—Ç—å—Å—è –Ω–µ–ø—Ä–∏–≤—ã—á–Ω—ã–º, –≤—ã –≤—Å–∫–æ—Ä–µ –∑–∞–º–µ—Ç–∏—Ç–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ —Å—Ö–æ–∂–µ—Å—Ç–µ–π. –í —ç—Ç–æ–º —Ä—É–∫–æ–≤–æ–¥—Å—Ç–≤–µ –ø–æ–¥—Ä–æ–±–Ω–æ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—é—Ç—Å—è –∏–∑–º–µ–Ω–µ–Ω–∏—è –∏ –æ–±—ä—è—Å–Ω—è–µ—Ç—Å—è, –∫–∞–∫ –≤—ã–ø–æ–ª–Ω–∏—Ç—å –æ–±–Ω–æ–≤–ª–µ–Ω–∏–µ. –¢–∞–∫–∂–µ –º—ã –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ–º –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ —Ç–æ–º, _–ø–æ—á–µ–º—É_ –±—ã–ª–∏ –≤–Ω–µ—Å–µ–Ω—ã —ç—Ç–∏ –∏–∑–º–µ–Ω–µ–Ω–∏—è.

## –ò–∑–º–µ–Ω–µ–Ω–∏—è –≤ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–µ —Ä–µ–∞–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏

–í –æ—Å–Ω–æ–≤–µ Svelte 5 –ª–µ–∂–∏—Ç –Ω–æ–≤—ã–π API —Ä—É–Ω. –†—É–Ω—ã –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —Å–æ–±–æ–π –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–∞, –∫–æ—Ç–æ—Ä—ã–µ —Å–æ–æ–±—â–∞—é—Ç Svelte –æ —Ä–µ–∞–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏. –°–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫–∏ —Ä—É–Ω—ã –≤—ã–≥–ª—è–¥—è—Ç –∫–∞–∫ —Ñ—É–Ω–∫—Ü–∏–∏, –Ω–∞—á–∏–Ω–∞—é—â–∏–µ—Å—è —Å —Å–∏–º–≤–æ–ª–∞ –¥–æ–ª–ª–∞—Ä–∞.

### let -> $state

–í Svelte 4 –æ–±—ä—è–≤–ª–µ–Ω–∏–µ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π —Å –ø–æ–º–æ—â—å—é `let` –Ω–∞ –≤–µ—Ä—Ö–Ω–µ–º —É—Ä–æ–≤–Ω–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ —Å—Ç–∞–Ω–æ–≤–∏–ª–æ—Å—å —Ä–µ–∞–∫—Ç–∏–≤–Ω—ã–º. –í Svelte 5 —ç—Ç–æ—Ç –ø—Ä–æ—Ü–µ—Å—Å —Å—Ç–∞–ª –±–æ–ª–µ–µ —è–≤–Ω—ã–º: –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è —Ä–µ–∞–∫—Ç–∏–≤–Ω–æ–π —Ç–æ–ª—å–∫–æ –ø—Ä–∏ —Å–æ–∑–¥–∞–Ω–∏–∏ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º —Ä—É–Ω—ã `$state`. –î–∞–≤–∞–π—Ç–µ –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ–º —Å—á—ë—Ç—á–∏–∫ –≤ —Ä–µ–∂–∏–º —Ä—É–Ω, –æ–±–µ—Ä–Ω—É–≤ –µ–≥–æ –≤ `$state`:

```diff lang="svelte"
<script>
-  let count = 0;
+  let count = $state(0);
</script>
```

–ù–∏—á–µ–≥–æ –±–æ–ª—å—à–µ –Ω–µ –º–µ–Ω—è–µ—Ç—Å—è. `count` –ø–æ-–ø—Ä–µ–∂–Ω–µ–º—É —è–≤–ª—è–µ—Ç—Å—è —Å–∞–º–∏–º —á–∏—Å–ª–æ–º, –∏ –≤—ã —á–∏—Ç–∞–µ—Ç–µ –∏ –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç–µ –µ–≥–æ –Ω–∞–ø—Ä—è–º—É—é, –±–µ–∑ –æ–±—ë—Ä—Ç–∫–∏, —Ç–∞–∫–æ–π –∫–∞–∫ `.value` –∏–ª–∏ `getCount()`.

:::tip[–ü–æ—á–µ–º—É –º—ã —Ç–∞–∫ —Å–¥–µ–ª–∞–ª–∏]
–ù–µ—è–≤–Ω–∞—è —Ä–µ–∞–∫—Ç–∏–≤–Ω–æ—Å—Ç—å `let` –Ω–∞ –≤–µ—Ä—Ö–Ω–µ–º —É—Ä–æ–≤–Ω–µ —Ä–∞–±–æ—Ç–∞–ª–∞ —Ö–æ—Ä–æ—à–æ, –Ω–æ —ç—Ç–æ –æ–≥—Ä–∞–Ω–∏—á–∏–≤–∞–ª–æ —Ä–µ–∞–∫—Ç–∏–≤–Ω–æ—Å—Ç—å ‚Äî –æ–±—ä—è–≤–ª–µ–Ω–∏–µ `let` –≤ –ª—é–±–æ–º –¥—Ä—É–≥–æ–º –º–µ—Å—Ç–µ –Ω–µ –±—ã–ª–æ —Ä–µ–∞–∫—Ç–∏–≤–Ω—ã–º. –≠—Ç–æ –∑–∞—Å—Ç–∞–≤–ª—è–ª–æ –≤–∞—Å –ø—Ä–∏–±–µ–≥–∞—Ç—å –∫ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—é —Ö—Ä–∞–Ω–∏–ª–∏—â –ø—Ä–∏ —Ä–µ—Ñ–∞–∫—Ç–æ—Ä–∏–Ω–≥–µ –∫–æ–¥–∞ –∑–∞ –ø—Ä–µ–¥–µ–ª–∞–º–∏ –≤–µ—Ä—Ö–Ω–µ–≥–æ —É—Ä–æ–≤–Ω—è –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤ –¥–ª—è –ø–æ–≤—Ç–æ—Ä–Ω–æ–≥–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è. –í —Ä–µ–∑—É–ª—å—Ç–∞—Ç–µ –≤–∞–º –ø—Ä–∏—Ö–æ–¥–∏–ª–æ—Å—å –æ—Å–≤–∞–∏–≤–∞—Ç—å —Å–æ–≤–µ—Ä—à–µ–Ω–Ω–æ –¥—Ä—É–≥—É—é –º–æ–¥–µ–ª—å —Ä–µ–∞–∫—Ç–∏–≤–Ω–æ—Å—Ç–∏, –∏ —Ä–∞–±–æ—Ç–∞—Ç—å —Å –Ω–µ–π —á–∞—Å—Ç–æ –±—ã–ª–æ –Ω–µ —Ç–∞–∫ —É–¥–æ–±–Ω–æ. –ü–æ—Å–∫–æ–ª—å–∫—É —Ä–µ–∞–∫—Ç–∏–≤–Ω–æ—Å—Ç—å –≤ Svelte 5 —Å—Ç–∞–ª–∞ –±–æ–ª–µ–µ —è–≤–Ω–æ–π, –≤—ã –º–æ–∂–µ—Ç–µ –ø—Ä–æ–¥–æ–ª–∂–∞—Ç—å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç–æ—Ç –∂–µ API –∏ –≤–Ω–µ –≤–µ—Ä—Ö–Ω–µ–≥–æ —É—Ä–æ–≤–Ω—è –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤. –ü–µ—Ä–µ–π–¥–∏—Ç–µ –∫ [—É—á–µ–±–Ω–∏–∫—É](https://svelte.dev/tutorial), —á—Ç–æ–±—ã —É–∑–Ω–∞—Ç—å –±–æ–ª—å—à–µ.
:::

### $: -> $derived/$effect

–í Svelte 4 –æ–ø–µ—Ä–∞—Ç–æ—Ä `$:` –Ω–∞ –≤–µ—Ä—Ö–Ω–µ–º —É—Ä–æ–≤–Ω–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª—Å—è –¥–ª—è –æ–±—ä—è–≤–ª–µ–Ω–∏—è –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–æ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏—è, —Ç–æ –µ—Å—Ç—å —Å–æ—Å—Ç–æ—è–Ω–∏—è, –∫–æ—Ç–æ—Ä–æ–µ –ø–æ–ª–Ω–æ—Å—Ç—å—é –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è —á–µ—Ä–µ–∑ –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ –¥—Ä—É–≥–∏—Ö —Å–æ—Å—Ç–æ—è–Ω–∏–π. –í Svelte 5 —ç—Ç–æ –¥–æ—Å—Ç–∏–≥–∞–µ—Ç—Å—è —Å –ø–æ–º–æ—â—å—é —Ä—É–Ω—ã `$derived`:

```diff lang="svelte"
<script>
-  let count = 0;
-  $: double = count * 2;
+  let count = $state(0);
+  const double = $derived(count * 2);
</script>
```

–ö–∞–∫ –∏ –≤ —Å–ª—É—á–∞–µ —Å–æ `$state`, –Ω–∏—á–µ–≥–æ –±–æ–ª—å—à–µ –Ω–µ –º–µ–Ω—è–µ—Ç—Å—è. `double` –ø–æ-–ø—Ä–µ–∂–Ω–µ–º—É —è–≤–ª—è–µ—Ç—Å—è —Å–∞–º–∏–º —á–∏—Å–ª–æ–º, –∏ –≤—ã —á–∏—Ç–∞–µ—Ç–µ –µ–≥–æ –Ω–∞–ø—Ä—è–º—É—é, –±–µ–∑ –æ–±—ë—Ä—Ç–∫–∏, —Ç–∞–∫–æ–π –∫–∞–∫ `.value` –∏–ª–∏ `getDouble()`.

–û–ø–µ—Ä–∞—Ç–æ—Ä `$:` —Ç–∞–∫–∂–µ –º–æ–∂–Ω–æ –±—ã–ª–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –ø–æ–±–æ—á–Ω—ã—Ö —ç—Ñ—Ñ–µ–∫—Ç–æ–≤. –í Svelte 5 —ç—Ç–æ –¥–æ—Å—Ç–∏–≥–∞–µ—Ç—Å—è —Å –ø–æ–º–æ—â—å—é —Ä—É–Ω—ã `$effect`:

```diff lang="svelte"
<script>
-  let count = 0;
-  $: {
+  let count = $state(0);
+  $effect(() => {
    if (count > 5) {
      alert('Count is too high!');
    }
-  }
+  });
</script>
```

–û–ë—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ [–ª–æ–≥–∏–∫–∞ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è `$effect` –æ—Ç–ª–∏—á–∞–µ—Ç—Å—è](/runes/effect#–ø–æ–Ω–∏–º–∞–Ω–∏–µ-–∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–µ–π) –æ—Ç –ª–æ–≥–∏–∫–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è `$:`.

:::tip[–ü–æ—á–µ–º—É –º—ã —Ç–∞–∫ —Å–¥–µ–ª–∞–ª–∏]
`$:` –±—ã–ª –æ—Ç–ª–∏—á–Ω—ã–º —Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ–º –∏ –ø—Ä–æ—Å—Ç—ã–º —Å–ø–æ—Å–æ–±–æ–º –Ω–∞—á–∞—Ç—å —Ä–∞–±–æ—Ç—É: –≤—ã –º–æ–≥–ª–∏ –ø—Ä–æ—Å—Ç–æ –¥–æ–±–∞–≤–∏—Ç—å `$:` –ø–µ—Ä–µ–¥ –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ–º –∫–æ–¥–∞, –∏ —ç—Ç–æ –∫–∞–∫-—Ç–æ —Ä–∞–±–æ—Ç–∞–ª–æ. –û–¥–Ω–∞–∫–æ —ç—Ç–∞ –∏–Ω—Ç—É–∏—Ç–∏–≤–Ω–æ—Å—Ç—å —Ç–∞–∫–∂–µ –±—ã–ª–∞ –µ–≥–æ –Ω–µ–¥–æ—Å—Ç–∞—Ç–∫–æ–º –ø–æ –º–µ—Ä–µ —É—Å–ª–æ–∂–Ω–µ–Ω–∏—è –∫–æ–¥–∞, —Ç–∞–∫ –∫–∞–∫ —Å—Ç–∞–ª–æ —Ç—Ä—É–¥–Ω–µ–µ —Ä–∞—Å—Å—É–∂–¥–∞—Ç—å –æ –µ–≥–æ –Ω–∞–º–µ—Ä–µ–Ω–∏—è—Ö. –ë—ã–ª–æ –ª–∏ —Ü–µ–ª—å—é –∫–æ–¥–∞ —Å–æ–∑–¥–∞–Ω–∏–µ –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–æ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏—è –∏–ª–∏ –ø–æ–±–æ—á–Ω–æ–≥–æ —ç—Ñ—Ñ–µ–∫—Ç–∞? –° `$derived` –∏ `$effect` –≤–∞–º –Ω—É–∂–Ω–æ –±—É–¥–µ—Ç –Ω–µ–º–Ω–æ–≥–æ –±–æ–ª—å—à–µ –ø–æ–¥—É–º–∞—Ç—å –∑–∞—Ä–∞–Ω–µ–µ (—Å–ø–æ–π–ª–µ—Ä: –≤ 90% —Å–ª—É—á–∞–µ–≤ –≤–∞–º –Ω—É–∂–µ–Ω `$derived`), –Ω–æ –≤–∞–º –∏ –¥—Ä—É–≥–∏–º —Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–∞–º –≤ –≤–∞—à–µ–π –∫–æ–º–∞–Ω–¥–µ –±—É–¥–µ—Ç –ª–µ–≥—á–µ —Ä–∞–∑–±–∏—Ä–∞—Ç—å—Å—è –≤ –∫–æ–¥–µ.

–¢–∞–∫–∂–µ –±—ã–ª–∏ –ø–æ–¥–≤–æ–¥–Ω—ã–µ –∫–∞–º–Ω–∏, –∫–æ—Ç–æ—Ä—ã–µ –±—ã–ª–æ —Ç—Ä—É–¥–Ω–æ –∑–∞–º–µ—Ç–∏—Ç—å:

- `$:` –æ–±–Ω–æ–≤–ª—è–ª—Å—è —Ç–æ–ª—å–∫–æ –Ω–µ–ø–æ—Å—Ä–µ–¥—Å—Ç–≤–µ–Ω–Ω–æ –ø–µ—Ä–µ–¥ —Ä–µ–Ω–¥–µ—Ä–∏–Ω–≥–æ–º, —á—Ç–æ –æ–∑–Ω–∞—á–∞–ª–æ, —á—Ç–æ –≤—ã –º–æ–≥–ª–∏ —á–∏—Ç–∞—Ç—å —É—Å—Ç–∞—Ä–µ–≤—à–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è –º–µ–∂–¥—É –ø–µ—Ä–µ—Ä–∏—Å–æ–≤–∫–∞–º–∏.
- `$:` –≤—ã–ø–æ–ª–Ω—è–ª—Å—è —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω —Ä–∞–∑ –∑–∞ —Ç–∏–∫, —á—Ç–æ –æ–∑–Ω–∞—á–∞–ª–æ, —á—Ç–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –º–æ–≥–ª–∏ –≤—ã–ø–æ–ª–Ω—è—Ç—å—Å—è —Ä–µ–∂–µ, —á–µ–º –≤—ã –¥—É–º–∞–µ—Ç–µ.
