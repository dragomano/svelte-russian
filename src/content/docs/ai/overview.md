Следующие страницы помогут вам настроить и использовать ИИ-инструменты, официально поддерживаемые командой Svelte.

Существует четыре инструмента, созданных для того, чтобы помочь вашему агенту писать корректный и надёжный код на Svelte. Они спроектированы для совместной работы, но каждый из них можно использовать отдельно:

- [Инструкции](/ai/instructions): небольшой промпт, который всегда добавляется в вашу сессию, чтобы агент лучше понимал доступные инструменты
- [MCP-сервер](/ai/mcp/overview): инструменты, промпты и ресурсы, которые дают агенту больше контекста — путём прямого обращения к официальной документации Svelte и использования статического анализа для исправления типичных ошибок генеративного ИИ
- [Навыки](/ai/skills): подгружаемые по требованию описания, которые обучают агента лучшим практикам Svelte, а также тому, как использовать [CLI `@sveltejs/mcp`](/ai/mcp/cli)
- [Субагенты](/ai/subagents): специализированные агенты, которых можно вызывать параллельно для выполнения атомарных операций в отдельном контекстном окне
