Открытый [репозиторий](https://github.com/sveltejs/ai-tools) с кодом MCP-сервера также является [маркетплейсом плагинов](https://code.claude.com/docs/ru/discover-plugins) для Claude Code.

Маркетплейс позволяет установить плагин `svelte`, который предоставит удалённый MCP-сервер, [навыки](/ai/skills) для инструктирования LLM о том, как правильно писать код на Svelte 5, и специализированного агента для редактирования Svelte-файлов.

По возможности рекомендуем инструктировать LLM выполнять MCP-вызовы через агента (можно явно упомянуть агента в сообщении, чтобы делегировать ему работу) при создании или редактировании файлов `.svelte` или модулей `.svelte.ts`/`.svelte.js` — это поможет экономить контекст за счёт более эффективной обработки задач, специфичных для Svelte.

## Установка

Чтобы добавить репозиторий как маркетплейс, запустите Claude Code и введите следующее:

```bash
/plugin marketplace add sveltejs/ai-tools
```

Затем установите плагин Svelte:

```bash
/plugin install svelte
```
